# Breadcrumb

Shows the current page location within a hierarchy.

> **Source:** Re-exported from `antd`. Full documentation: [Ant Design Breadcrumb](https://ant.design/components/breadcrumb/)

## Basic Usage

```tsx
import { Breadcrumb } from '@vef-framework-react/components';
```

For complete usage examples and API documentation, refer to the [Ant Design Breadcrumb documentation](https://ant.design/components/breadcrumb/).
