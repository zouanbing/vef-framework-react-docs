# XAttachments

File attachment area for AI chat interfaces, supporting drag-and-drop and file preview.

> **Source:** Re-exported from `@ant-design/x`. Full documentation: [Ant Design X Attachments](https://x.ant.design/components/attachments)

## Basic Usage

```tsx
import { XAttachments } from '@vef-framework-react/components';

export default function Demo() {
  return (
    <XAttachments
      items={[]}
      onChange={(info) => console.log(info)}
    />
  );
}
```

> For the full API, see [Ant Design X Attachments docs](https://x.ant.design/components/attachments).
