# Affix 固钉

将页面元素钉在可视范围。

> **来源：** 直接从 `antd` 重新导出。完整文档：[Ant Design Affix](https://ant.design/components/affix-cn/)

## 基础用法

```tsx
import { Affix } from '@vef-framework-react/components';
```

完整用法示例和 API 文档请参阅 [Ant Design Affix 中文文档](https://ant.design/components/affix-cn/)。
