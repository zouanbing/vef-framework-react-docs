# Anchor 锚点

用于跳转到页面指定位置。

> **来源：** 直接从 `antd` 重新导出。完整文档：[Ant Design Anchor](https://ant.design/components/anchor-cn/)

## 基础用法

```tsx
import { Anchor } from '@vef-framework-react/components';
```

完整用法示例和 API 文档请参阅 [Ant Design Anchor 中文文档](https://ant.design/components/anchor-cn/)。
