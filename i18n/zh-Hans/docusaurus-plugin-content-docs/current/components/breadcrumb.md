# Breadcrumb 面包屑

显示当前页面在系统层级中的位置。

> **来源：** 直接从 `antd` 重新导出。完整文档：[Ant Design Breadcrumb](https://ant.design/components/breadcrumb-cn/)

## 基础用法

```tsx
import { Breadcrumb } from '@vef-framework-react/components';
```

完整用法示例和 API 文档请参阅 [Ant Design Breadcrumb 中文文档](https://ant.design/components/breadcrumb-cn/)。
