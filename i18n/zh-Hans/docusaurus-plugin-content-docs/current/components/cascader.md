# Cascader 级联选择

从一组相关联的数据集合中进行选择。

> **来源：** 直接从 `antd` 重新导出。完整文档：[Ant Design Cascader](https://ant.design/components/cascader-cn/)

## 基础用法

```tsx
import { Cascader } from '@vef-framework-react/components';
```

完整用法示例和 API 文档请参阅 [Ant Design Cascader 中文文档](https://ant.design/components/cascader-cn/)。
