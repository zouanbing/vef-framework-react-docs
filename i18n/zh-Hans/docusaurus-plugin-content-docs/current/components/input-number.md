# InputNumber 数字输入框

通过鼠标或键盘输入范围内的数值。

> **来源：** 直接从 `antd` 重新导出。完整文档：[Ant Design InputNumber](https://ant.design/components/input-number-cn/)

## 基础用法

```tsx
import { InputNumber } from '@vef-framework-react/components';
```

完整用法示例和 API 文档请参阅 [Ant Design InputNumber 中文文档](https://ant.design/components/input-number-cn/)。
