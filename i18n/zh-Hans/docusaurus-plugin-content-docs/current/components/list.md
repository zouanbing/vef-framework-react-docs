# List 列表

最基础的列表展示，可承载文字、列表、图片、段落。

> **来源：** 直接从 `antd` 重新导出。完整文档：[Ant Design List](https://ant.design/components/list-cn/)

## 基础用法

```tsx
import { List } from '@vef-framework-react/components';
```

完整用法示例和 API 文档请参阅 [Ant Design List 中文文档](https://ant.design/components/list-cn/)。
