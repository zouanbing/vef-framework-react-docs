# Menu 导航菜单

为页面和功能提供导航的菜单列表。

> **来源：** 直接从 `antd` 重新导出。完整文档：[Ant Design Menu](https://ant.design/components/menu-cn/)

## 基础用法

```tsx
import { Menu } from '@vef-framework-react/components';
```

完整用法示例和 API 文档请参阅 [Ant Design Menu 中文文档](https://ant.design/components/menu-cn/)。
