# Popover 气泡卡片

点击/鼠标移入元素，弹出气泡式的卡片浮层。

> **来源：** 直接从 `antd` 重新导出。完整文档：[Ant Design Popover](https://ant.design/components/popover-cn/)

## 基础用法

```tsx
import { Popover } from '@vef-framework-react/components';
```

完整用法示例和 API 文档请参阅 [Ant Design Popover 中文文档](https://ant.design/components/popover-cn/)。
