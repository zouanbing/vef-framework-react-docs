# Radio 单选框

单选框，用于在多个备选项中选中单个状态。

> **来源：** 直接从 `antd` 重新导出。完整文档：[Ant Design Radio](https://ant.design/components/radio-cn/)

## 基础用法

```tsx
import { Radio } from '@vef-framework-react/components';
```

完整用法示例和 API 文档请参阅 [Ant Design Radio 中文文档](https://ant.design/components/radio-cn/)。
