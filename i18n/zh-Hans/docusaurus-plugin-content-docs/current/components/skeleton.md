# Skeleton 骨架屏

在需要等待加载内容的位置提供一个占位图形组合。

> **来源：** 直接从 `antd` 重新导出。完整文档：[Ant Design Skeleton](https://ant.design/components/skeleton-cn/)

## 基础用法

```tsx
import { Skeleton } from '@vef-framework-react/components';
```

完整用法示例和 API 文档请参阅 [Ant Design Skeleton 中文文档](https://ant.design/components/skeleton-cn/)。
