# Slider 滑动输入条

滑动型输入器，展示当前值和可选范围。

> **来源：** 直接从 `antd` 重新导出。完整文档：[Ant Design Slider](https://ant.design/components/slider-cn/)

## 基础用法

```tsx
import { Slider } from '@vef-framework-react/components';
```

完整用法示例和 API 文档请参阅 [Ant Design Slider 中文文档](https://ant.design/components/slider-cn/)。
