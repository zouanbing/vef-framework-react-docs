# Watermark 水印

给页面的某个区域加上水印。

> **来源：** 直接从 `antd` 重新导出。完整文档：[Ant Design Watermark](https://ant.design/components/watermark-cn/)

## 基础用法

```tsx
import { Watermark } from '@vef-framework-react/components';
```

完整用法示例和 API 文档请参阅 [Ant Design Watermark 中文文档](https://ant.design/components/watermark-cn/)。
