# XNotification

Notification utility for AI events (re-exported from `@ant-design/x`).

> **Source:** Re-exported from `@ant-design/x` as `XNotification`. Full documentation: [Ant Design X](https://x.ant.design/)

## Basic Usage

```tsx
import { XNotification } from '@vef-framework-react/components';

XNotification.open({
  message: 'AI Response Ready',
  description: 'Your analysis has been completed.',
});
```

> For the full API, see [Ant Design X notification docs](https://x.ant.design/).
