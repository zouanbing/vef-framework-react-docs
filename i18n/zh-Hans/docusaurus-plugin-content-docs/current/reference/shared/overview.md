# Shared 包总览

`@vef-framework-react/shared` 是整个框架的“通用能力底座”。  
它主要包含四类导出:

1. 类型工具
2. 日期 / 格式化 / 校验
3. 事件、对象、树、路径、字符串等实用函数
4. 颜色与拼音等偏业务前端能力

## 推荐阅读顺序

1. [类型工具](./types.md)
2. [校验、时间、事件与安全](./chrono-event-validation.md)
3. [树、路径与数据结构](./data-and-structure.md)
4. [通用工具与颜色能力](./general-utilities.md)

## 这组文档怎么组织

- 高频、带明确语义的能力单独解释
- 超大批量的通用函数按模块分组，用表格逐项说明场景
